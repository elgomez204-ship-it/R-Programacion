# 005-Aserciones

## Introducción
La carpeta `005-Aserciones` contiene prácticas centradas en el uso de aserciones para comprobar condiciones dentro de un programa en Python.  
Estas actividades ayudan a verificar que el código se comporte como se espera durante su ejecución.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Tema central:** uso de la instrucción `assert` para validar condiciones.
- **Contenidos trabajados:** comprobación de valores, detección temprana de errores lógicos y validación de supuestos del programa.
- **Metodología:** ejercicios cortos orientados a depurar y mejorar la confiabilidad del código.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Aprender a validar condiciones clave dentro del programa.
- Detectar fallos lógicos de forma temprana durante el desarrollo.
- Mejorar la calidad del código comprobando supuestos importantes.
- Facilitar la depuración al identificar rápidamente dónde y por qué falla una condición.

## Conclusión
Las prácticas de `005-Aserciones` fortalecen el pensamiento crítico y la revisión del código al exigir comprobaciones explícitas de lo que debe cumplirse.  
El uso adecuado de aserciones contribuye a construir programas más correctos, claros y fáciles de mantener.
