# 002-Estructuras de repetición

## Introducción
La carpeta `002-Estructuras de repetición` contiene prácticas orientadas a repetir instrucciones de forma controlada en Python.  
En esta sección se trabaja cómo automatizar tareas mediante ciclos, evitando repetir código manualmente.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Tema central:** bucles o ciclos de repetición `for` y `while`.
- **Aplicación práctica:** recorridos por rangos de valores, conteos, acumulación de resultados y resolución de ejercicios repetitivos.
- **Archivos de práctica:** ejercicios progresivos, incluyendo problemas aplicados como `07-ejercicio muñecos.py`.
- **Metodología:** práctica paso a paso para controlar iteraciones, límites y condiciones de repetición.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Entender cómo ejecutar una misma acción múltiples veces de manera eficiente.
- Resolver problemas de conteo, acumulación y recorridos de datos.
- Mejorar la lógica de programación al definir inicios, finales y condiciones de parada en ciclos.
- Construir la base para algoritmos más complejos que requieren procesamiento repetitivo.

## Conclusión
Las prácticas de `002-Estructuras de repetición` desarrollan una habilidad esencial: automatizar procesos mediante ciclos.  
Gracias a estos ejercicios, se fortalece la capacidad de crear soluciones más rápidas, ordenadas y escalables en programación.
