# 002-Utilización de objetos

## Introducción
La carpeta `002-Utilización de objetos` reúne prácticas orientadas al uso de objetos dentro de la programación. En esta etapa se trabaja cómo representar entidades del problema mediante estructuras que agrupan datos y comportamiento, fortaleciendo la comprensión de un enfoque más organizado del código.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Enfoque principal:** uso y manipulación de objetos en ejercicios prácticos.
- **Organización:** archivos `.py` independientes por práctica o tema.
- **Competencias trabajadas:** creación de objetos, acceso a atributos, uso de métodos y aplicación en problemas sencillos.
- **Metodología:** aprendizaje incremental, con ejemplos concretos y progresivos.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Comprender cómo modelar elementos del mundo real mediante objetos.
- Practicar la relación entre datos atributos y acciones métodos.
- Escribir código más claro, reutilizable y fácil de mantener.
- Preparar la base para temas posteriores como estructuras más complejas y programación orientada a objetos a mayor profundidad.

## Conclusión
El trabajo de esta carpeta permite pasar de ejercicios básicos a una forma más estructurada de programar.  
Las prácticas de utilización de objetos refuerzan la lógica, la organización del código y la capacidad de resolver problemas con modelos más cercanos a situaciones reales.
