# 001-Estructuras de selección

## Introducción
La carpeta `001-Estructuras de selección` reúne prácticas enfocadas en la toma de decisiones dentro de un programa en Python.  
Aquí se trabajan distintos escenarios donde el código debe evaluar condiciones para ejecutar acciones diferentes según cada caso.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Tema central:** estructuras condicionales `if`, `else`, `elif` y anidación de condiciones.
- **Tipo de lógica aplicada:** comparaciones, evaluación booleana y selección de rutas de ejecución.
- **Metodología:** ejercicios cortos y progresivos para dominar decisiones en código.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Comprender cómo un programa decide qué acción ejecutar según una condición.
- Aprender a construir reglas lógicas con `if`, `else` y `elif`.
- Practicar escenarios con múltiples decisiones y condiciones anidadas.
- Preparar la base para resolver problemas más complejos con control de flujo.

## Conclusión
Las prácticas de `001-Estructuras de selección` fortalecen la lógica de programación al enseñar cómo controlar el comportamiento del programa según diferentes situaciones.  
En conjunto, esta carpeta consolida una habilidad clave para desarrollar soluciones más completas, claras y eficientes.
