# 001-Identificación de los elementos de un programa informático

## Introducción
La carpeta `001-Identificación de los elementos de un programa informático` contiene prácticas iniciales de Python orientadas a comprender los conceptos básicos de programación. Su estructura está dividida por temas para avanzar de forma progresiva, desde la creación del primer programa hasta el uso de operadores y expresiones.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Organización:** subcarpetas temáticas con scripts `.py` independientes.
- **Temas incluidos:** estructura y bloques fundamentales, variables, tipos de datos, literales, constantes y operadores.
- **Tipo de ejercicios:** prácticas cortas de entrada/salida, asignación de datos, conversiones y resolución de expresiones.
- **Enfoque:** aprendizaje práctico, un concepto por archivo.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Identificar y comprender los componentes esenciales de un programa.
- Aprender a declarar variables, manejar tipos de datos y usar constantes.
- Aplicar operadores aritméticos, de comparación y lógicos en ejercicios concretos.
- Desarrollar una base sólida para continuar con temas más avanzados de programación.

## Conclusión
El contenido de esta carpeta permite construir fundamentos técnicos de forma ordenada y clara. Cada práctica aporta una habilidad específica y, en conjunto, forma una ruta de aprendizaje útil para iniciar en programación y mejorar la lógica de resolución de problemas.
