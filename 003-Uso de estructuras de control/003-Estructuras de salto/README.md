# 003-Estructuras de salto

## Introducción
La carpeta `003-Estructuras de salto` contiene prácticas orientadas al uso de funciones para organizar y reutilizar código en Python.  
En esta etapa se trabaja cómo definir, invocar y estructurar bloques de instrucciones que permiten "saltar" de una parte del programa a otra de forma ordenada.

## Aspectos técnicos
- **Lenguaje de programación:** Python.
- **Tema central:** funciones y transferencia de control del programa.
- **Contenidos trabajados:** definición de funciones, parámetros, múltiples parámetros y valores de retorno.
- **Metodología:** ejercicios progresivos para comprender modularidad y reutilización de código.

## ¿Para qué sirve?
Esta carpeta sirve para:
- Aprender a dividir un programa en partes pequeñas y reutilizables.
- Mejorar la organización y legibilidad del código.
- Evitar repetir instrucciones mediante funciones que se pueden llamar varias veces.
- Comprender cómo enviar datos a una función y cómo recibir resultados.

## Conclusión
Las prácticas de `003-Estructuras de salto` fortalecen una base fundamental de programación: la modularidad.  
Con el uso de funciones, el código se vuelve más claro, mantenible y escalable, facilitando la resolución de problemas más complejos.
